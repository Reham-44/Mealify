## ✨ Features

- **Responsive Design** — Looks great on all screen sizes
- **Hero Section** — Eye-catching headline with a call-to-action to book a table
- **Meet the Chefs** — Showcase of professional chefs with photos and bios
- **Food Gallery** — Beautiful grid of signature dishes
- **Contact Section** — Address, email, phone, and opening hours
- **Newsletter Signup** — Footer subscription form
- **Smooth Navigation** — Anchor-based navigation with smooth scrolling

## 🚀 Getting Started

No dependencies or build tools needed — just open and go.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Reham-44/Mealify.git

# Navigate into the project
cd Mealify

# Open in your browser
open index.html
```

Or simply drag `index.html` into any browser window.
